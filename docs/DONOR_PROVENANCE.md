# Donor Provenance

Hanimo WebUI was reviewed alongside the author's private predecessor web,
CLI, editor, and desktop projects. The release uses clean-room adaptations of
behavior-level patterns: provider routing, retry handling, practical self-host
installation, and document-aware chat interaction.

The stable release does not copy internal endpoints, company SSO, private
prompts/data, organization branding, credentials, or mixed-author implementation
code. Private predecessor repositories are reference-only for this release and
are not release dependencies.
